# Python Mini Projects Collection

A collection of Python projects developed as part of academic coursework and practical learning in Cyber Security, Digital Forensics, Machine Learning, and Data Structures & Algorithms. Each project demonstrates the application of core computer science concepts through interactive and real-world implementations.

---

## Projects Included

### 1. Image Forgery Detection (CSDF)

A Digital Forensics application that detects image tampering using Error Level Analysis (ELA), edge detection, and EXIF metadata analysis. The project provides a graphical interface for verifying image authenticity.

**Technologies:** Python, OpenCV, Pillow, NumPy, Tkinter

---

### 2. File Integrity Verification using SHA-256 (CSDF2)

A Cyber Security project that verifies file integrity using the SHA-256 hashing algorithm. It generates secure hashes, detects unauthorized file modifications, and produces forensic reports.

**Technologies:** Python, hashlib, JSON, Tkinter

---

### 3. Multithreaded Merge Sort Visualization (DAA)

A visualization tool demonstrating Merge Sort and Multithreaded Merge Sort. The project compares execution times and illustrates the benefits of parallel processing through an interactive GUI.

**Technologies:** Python, Tkinter, Threading

---

### 4. Titanic Survival Prediction (ML)

A Machine Learning application that predicts passenger survival on the Titanic dataset using a Random Forest classifier. It includes model training and a graphical interface for real-time predictions.

**Technologies:** Python, Scikit-learn, Pandas, NumPy, Tkinter

---

### 5. Advanced Titanic Machine Learning Pipeline (ML_Project_Odd_Sem)

A comprehensive machine learning workflow featuring data preprocessing, feature engineering, model training, cross-validation, and evaluation using multiple classification algorithms including Logistic Regression, Random Forest, and XGBoost.

**Technologies:** Python, Pandas, NumPy, Scikit-learn, XGBoost, SHAP

## Requirements

Each project contains its own `requirements.txt` file with the required dependencies.

Install dependencies for a specific project using:

```bash
pip install -r requirements.txt
```

---

## How to Run

Navigate to the desired project folder and execute its main Python file.

Example:

```bash
cd csdf
python csdf_mini.py
```

or

```bash
cd ML
python Titanic_app.py
```

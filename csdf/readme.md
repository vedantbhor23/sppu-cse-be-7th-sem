# Image Forgery Detection using Digital Forensics

## Overview

This project is a Cyber Security and Digital Forensics application that detects possible image tampering using multiple forensic analysis techniques. It provides an intuitive graphical interface for analyzing digital images and identifying signs of manipulation.

## Features

- Error Level Analysis (ELA)
- Edge Detection
- EXIF Metadata Extraction
- Metadata Comparison
- Image Integrity Analysis
- User-friendly Tkinter GUI

## Technologies Used

- Python
- OpenCV
- Pillow (PIL)
- Tkinter
- NumPy
- EXIF Processing Libraries
